Apa itu github?

GitHub adalah platform pengembangan software online berbasis cloud yang digunakan untuk menyimpan, melacak, dan sebagai tempat kolaborasi antar developer dalam suatu proyek perangkat lunak.
Lebih singkatnya, github itu adalah sebuah platform untuk membuat blog.
Apa itu blog?
jadi, secara singkat, blog adalah sebuah situs yang berisi tulisan pada halaman web, sebenarnya blog itu ada banyak, cuman, kitanya aja yang gak tau apa itu blog, makanya disini, kita bakalan ngasih tau kalian.
